# :card_index_dividers: project-ditto

> Mock server

![alt](img/ditto.gif)

## :gear: Requisitos

* python 3
* virtualenv

## :tada: Uso

```bash
# Creacion del virtual env
virtualenv -p python3 env

# Activar el virtual env
. env/bin/activate

# Instalacion de los requerimientos
pip install -r requirements.txt

# Ejecutar
python app.py
```
